# SearchToo por Davii.

## Como funciona:
O SearchToo é uma ferramenta de pesquisa com a Wikipédia.

## Como usar:
Opção 1 (VSCode): Baixe em [Code] > [Download ZIP] > baixe o VS Code (https://code.visualstudio.com/download) > abra o app Visual Studio Code > instale a extensão Live Server de Ritwick Dey > clique em [Go Live] > Pronto!

Opção 2 (Python3) para macOS 12.3 ou superior: Baixe em [Code] > [Download ZIP] > abra o .zip instalado > selecione a pasta instalada > clique com os 2 botões do trackpad/botão direito do mouse > selecione [Nova aba de terminal na pasta] > digite: python3 --version > se der erro, digite: brew install python e se der certo, digite: python3 -m http.server 8470 > acesse http://localhost:8470 no navegador (Google Chrome, Safari, Firefox, Samsung Internet, etc).

## Este projeto usa:
HTML, CSS, JS, JSON, ICO e PNG.
